# Zoo

Zoo is a Typescript GraphQL API for Internet-of-Things devices, it is intended as a personal project for now.

## Installation

Clone the repo and install dependencies.

```bash
git clone https://github.com/shadak/Zoo
cd Zoo
yarn
```
